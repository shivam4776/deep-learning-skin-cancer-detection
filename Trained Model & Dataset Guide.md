## 📥 Pretrained Model & Dataset

If you would like to test the project without training the model from scratch, you can download the pretrained model from the Google Drive link below:

🔗 **Pretrained Model:**
[https://drive.google.com/drive/folders/1G7bXSM5mEV_LUZuArBcsifcQYePfIJqV?usp=drive_link]

The model was trained on the ISIC Skin Cancer Dataset and achieved **90.01% classification accuracy** across **9 skin disease categories**.

### 📊 Dataset

The dataset used for training can be downloaded from Kaggle:

🔗 **Kaggle Dataset:**
[https://www.kaggle.com/datasets/nodoubttome/skin-cancer9-classesisic]

### ▶️ How to Use the Pretrained Model

1. Download the pretrained model from Google Drive.
2. Place the model file inside the `saved_model/` directory.
3. Install the required dependencies:

   ```bash
   pip install -r requirements.txt
   ```
4. Run the inference or prediction script to classify skin lesion images.
